2. Project Details

2.1 Abstract

The abstract of the project should be written in English, since this text will be published to demonstrate ongoing work on Lichtenberg computer. It should consist of about 300 words. Typically, this abstract will be published by LRZ on the project web pages (see also 5.)

2.2 Detailed Description/Report

(This section may be omitted if you just apply for a test account.)

A detailed description of the project is supposed to outline the state of the art, the procedures used, and references of work already accomplished. The following topics should be included in your description:
  • research field (e.g. Chemistry),
  • underlying theory (e.g. Quantum Mechanics),
  • description of goals (e.g. Computation of Helicopter Aerodynamics),
  • mathematical model (e.g. Navier-Stokes-Equations with turbulence model),
  • numerical methods (e.g. Finite Volume Method with Multigrid on unstructured tetrahedron grids,
  • state of the art,
  • calculation and rationale of resources ( CPU-time, memory number of grid nodes, number of time steps, ...),
  • expected performance and scaling behaviour,
  • practical relevance of expected results, time frame and work plan,
  • necessity for use of high performance computing,
  • experiences of the proposer with HPC-technologies, previous results and/or limits from other platforms,
  • relevant literature or publications.

This description is to be sent in separate file. You may upload that file after you have completed and successfully submitted this proposal form, using the upload form at https://www.lrz.de/services/compute/supermuc/upload-report.

3.5 Test Accounts

The steering committee grants test accounts with restricted resources before the project is finally reviewed. Those test accounts are granted for any project proposal that comes with a sufficiently explanatory abstract and/or detailed description.

However, the risk of useless work in case the project is rejected remains with the project proposer.

The duration of a testproject is at most one year.
